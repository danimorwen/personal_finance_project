# WalletCare Project

This project was made during the PYSTACK WEEK 7.0 and consist of an app for personal financial management.

## Features

### Home
* Accounts balance
* Income and expenses balance
* Monthly income and expenses
* Link to account management
* Link to add new transactions
* Link to statements
* Link to planning creation and visualization
* Financial health visualization
* Link to category edition
* Link to invoice creation and visualization
* Statistics
